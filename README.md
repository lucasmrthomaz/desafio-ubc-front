# Desafio UBC - .NET6 + React

Tecnologias envolvidas
- ReactJS + NEXT.JS 14
- Bootstrap 5 , Fontawesome
- Jest (tests)

# Rodar localmente
- npm install --force
> Esteja com o backend rodando
- npm start
> O projeto ira iniciar na porta 3000
